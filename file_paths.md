# Путь к файлу в Windows

**Путь к файлу** — это строка, указывающая местоположение файла или папки в файловой системе.

Пример: `C:\Users\Student\Documents`

- **C:** — буква диска (корневой каталог)
- **Users** — папка, содержащая профили пользователей
- **Student** — имя конкретного пользователя
- **Documents** — папка с документами пользователя

Пути могут быть:
- **Абсолютные** — полный путь от корня диска (например, `C:\Windows\System32\drivers\etc\hosts`)
- **Относительные** — путь относительно текущей папки (например, `..\Documents\file.txt`)

В Windows используется обратная косая черта `\` в качестве разделителя.
