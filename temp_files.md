# Временные файлы Windows

Временные файлы (.tmp, .temp, файлы в папках Temp) создаются программами и системой для:
- Хранения промежуточных данных при работе (например, кэш браузеров)
- Обеспечения отказоустойчивости (автосохранение в офисных приложениях)
- Установки обновлений и драйверов

### Почему их нужно контролировать?
- Они занимают дисковое пространство (иногда до десятков ГБ)
- Могут замедлять работу системы при переполнении диска
- Некоторые временные файлы могут содержать конфиденциальные данные

### Как очистить временные файлы?
1. Встроенная утилита **Очистка диска** (cleanmgr)
2. Настройка **Хранилища** в параметрах Windows 10
3. Ручная очистка папок:
   - `%TEMP%` и `%WINDIR%\Temp`
   - `C:\Windows\Prefetch`
   - Кэш браузеров
