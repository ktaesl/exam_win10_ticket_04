# Структура папок и временные файлы Windows 10

Репозиторий для экзаменационного билета №4 по дисциплине "Конфигурирование Windows 10".

В данном проекте рассматриваются:
- Основные системные папки Windows 10
- Пути к файлам и их структура
- Роль временных файлов и способы их очистки
- Правила безопасной работы с файлами и папками
